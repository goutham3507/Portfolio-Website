# 📌 Portfolio Website

## 🌟 Overview
This is a **fully responsive portfolio website** built with **React.js, Tailwind CSS, and GSAP**, designed to showcase projects, skills, and experience in a visually engaging and interactive way. It includes smooth animations, a sleek UI, and optimized performance for both desktop and mobile devices.

## 🚀 Features
- 🎨 **Modern UI/UX** – Clean and minimal design with a dark theme.
- 🖥️ **Responsive Design** – Optimized for mobile, tablet, and desktop.
- 🔥 **GSAP Animations** – Smooth transitions and scrolling effects.
- 📄 **Downloadable Resume** – Easy access to a personal resume.
- 🛠️ **Projects Showcase** – Displays work with tech stack details.

## 🛠 Tech Stack
| Technology  | Purpose |
|------------|---------|
| **React.js** | Frontend framework |
| **Tailwind CSS** | Styling and responsive design |
| **GSAP** | Smooth animations and transitions |

## 🔧 Installation & Setup
1. **Clone the Repository:**
   ```bash
   git clone https://github.com/goutham3507/Portfolio-Website.git
   cd Portfolio-Website
   ```
2. **Install Dependencies:**
   ```bash
   npm install
   ```
3. **Run the Development Server:**
   ```bash
   npm run dev
   ```
4. Open **http://localhost:5173/** to view it in the browser.

## 🌍 Deployment
This portfolio is deployed using **Vercel**. To deploy:
```bash
vercel --prod
```
